# 🦏 BIG PET

ฟีเจอร์ **Big Pet Feeding** มีโหมดหลัก ๆ ให้เลือกใช้งาน 4 แบบ ได้แก่:

#### 🔹 1. Amount

🍎 ให้อาหารตามจำนวนที่กำหนด

* เมื่อครบตามจำนวนที่ตั้งไว้ จะหยุดให้อัตโนมัติ
* หากใส่ `-1` = ให้อาหารแบบไม่จำกัด

#### 🔹 2. Max Level

📈 ให้อาหารอัตโนมัติจนกว่า **Big Pet จะถึงเลเวลสูงสุด**

#### 🔹 3. Skin

🎨 ใช้กับอาหารที่เปลี่ยน **สกินสัตว์** (เช่น สัตว์น้ำ, ไดโนเสาร์)

* จะให้อาหารเรื่อยๆจนกว่าจะได้ **สกินที่เลือกไว้**
* เมื่อได้แล้ว จะหยุดให้อัตโนมัติ

#### 🔹 4. Mutation

❄️🔥 ใช้กับอาหารที่ทำให้สัตว์ได้ **บัฟ (Mutation Buff)** เช่น Snow, Jurassic

* จะให้อาหารเรื่อย ๆ จนกว่าจะได้บัฟที่ต้องการ
* เมื่อได้แล้ว จะหยุดให้อัตโนมัติ

***

### ⚙️ วิธีใช้งาน

1. เลือกโหมดที่ต้องการ (Amount, Max Level, Skin, Mutation)
2. เลือกว่าให้อาหารกับ **BIG PET 1, BIG PET 2 หรือ BIG PET 3**
3. กดปุ่ม **Create Rule**

{% hint style="success" %}
* **BIG PET 1** → 🦖 สัตว์ตัวแรกหน้าฟาร์ม ที่ซื้อได้ด้วย **เงินในเกม 30K**
* **BIG PET 2** → 🐉 สัตว์ตัวที่สองหน้าฟาร์ม ซื้อด้วย **Robux**
* **BIG PET 3** → 🐟 สัตว์น้ำที่อยู่ **ด้านหลังฟาร์ม**
{% endhint %}

***

### 📌 สิ่งที่ต้องตั้งค่าเพิ่มหลังจากกด Create Rule

* **Amount** → เลือกอาหาร + จำนวนที่จะให้ (`-1` = ไม่จำกัด)
* **Max Level** → เลือกอาหารที่จะใช้ให้อัตโนมัติ
*   **Mutation** → เลือกอาหาร + บัฟที่ต้องการ (เลือกให้ถูกว่าอาหารไหนสุ่มบัฟได้)\
