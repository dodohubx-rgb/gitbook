# 🕹️ Play mode

### 🔁 Auto Swap

ฟังก์ชันนี้ใช้สำหรับ **เปลี่ยนสัตว์อัตโนมัติ (Auto Swap) เมื่อขี่ไปใกล้ๆ**

#### ⚙️ การตั้งค่า

* **Animal:**\
  เลือกชื่อสัตว์ที่ต้องการให้ระบบเปลี่ยนอัตโนมัติ
* **Enabled:**\
  เปิด (✅) เพื่อให้ระบบเปลี่ยนสัตว์โดยอัตโนมัติเมื่อถึงเงื่อนไขที่กำหนด

***

### 💀 Auto Retry

ฟังก์ชันนี้ช่วยให้ **ระบบเริ่มใหม่โดยอัตโนมัติเมื่อคุณตาย (Auto Retry)**

#### ⚙️ การตั้งค่า

* **Enabled:**\
  เปิด (✅) เพื่อให้ระบบรีสตาร์ทโดยอัตโนมัติเมื่อตาย

***

### 🦁 Lion Run (BETA, Unstable)

โหมดนี้ใช้ **AI Pathfinding** เพื่อช่วยให้สิงโตหลบสิ่งกีดขวางโดยอัตโนมัติ

#### ⚙️ การตั้งค่า

*   **Enabled:**\
    เปิด (✅) เพื่อเปิดใช้งานโหมด Lion Run

    > 🧠 ต้องใช้สัตว์ประเภท **Lion** เท่านั้น\
    > 💻 แนะนำให้ใช้กับอุปกรณ์สเปกสูง

💡 **คำเตือน:**\
ฟีเจอร์นี้ยังอยู่ใน **BETA (ทดสอบ)** อาจมีความไม่เสถียรในบางครั้ง
